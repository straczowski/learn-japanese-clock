# Learn Japanese Clock

Interactive web app to practice telling time in Japanese. Type the time in Hiragana and get instant feedback with audio pronunciation.

 **[Try it live](https://straczowski.github.io/learn-japanese-clock/)**

## Quick Start

```bash
npm install
npm run dev
```

## Setup

Audio files use Git LFS. After cloning:

```bash
git lfs install
git lfs pull
```
